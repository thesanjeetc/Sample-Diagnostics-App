# Sample Diagnostics App
## Install Extension
- Load unpacked folder 'extension'.

## Online Demo
- Visit [https://sample-diagnostics.vercel.app](https://sample-diagnostics.vercel.app/data).

## Running Locally
- `cd app`
- `npm install`
- `npm start`
- Visit [http://localhost:3000](http://localhost:3000/).
